1. The Misconception

Ask a machine learning student about the difference between discriminative and generative models and the answer will often be:

“Discriminative models classify data, while generative models generate data.”

Although this statement is not entirely incorrect, it hides the fundamental distinction between the two paradigms and often leads to a shallow understanding of what these models actually learn.

Consider the following examples:

  • Logistic Regression is a discriminative model.
  • Support Vector Machines are discriminative models.
  • Naive Bayes is a generative model.
  • Gaussian Discriminant Analysis is a generative model.
  • Large Language Models are generative models.

At first glance, the distinction seems obvious. Logistic Regression classifies emails as spam or not spam, while a Large Language Model generates text. End of story.

However, this interpretation immediately raises an interesting question.

If a model can generate realistic examples from a class, shouldn't it also be able to recognize that class?

A generative model learns class distributions, so it can classify using Bayes rule and generate new samples by sampling. A discriminative model learns the boundary needed for classification.

The figure above already hints at a deeper asymmetry. A model that learns how data is generated may naturally acquire the ability to classify. In contrast, a model that only learns a classification rule does not necessarily learn anything about how the data itself was produced.

In fact, many generative models can perform classification remarkably well. Classical examples include Naive Bayes and Gaussian Discriminant Analysis.

Now consider the opposite direction.

If a model can perfectly classify data, does it know how that data was generated?

Surprisingly, the answer is generally no.

A classifier may learn an almost perfect decision boundary while having little understanding of the underlying process that produced the data.

This observation reveals that the true distinction between discriminative and generative models is not about classification versus generation. Instead, it is about the amount of information each model attempts to learn about the world.
