4. Information Learned

The experiment revealed an asymmetry: a generative model can derive a classifier, but a discriminative model cannot generally derive a generator.

Which object contains more information: \(P(y|x)\) or \(P(x,y)\)?

A discriminative model learns the conditional distribution:

$$ P(y|x) $$

This is exactly what we need for classification. But it says nothing about how likely the observation \(x\) itself is.

A generative model learns the joint distribution:

$$ P(x,y) = P(y|x)P(x) = P(x|y)P(y) $$

This object contains both the relationship between observations and labels, and the distribution of the observations themselves.

Knowing \(P(x,y)\) lets us recover \(P(y|x)\), \(P(x|y)\), \(P(x)\), and \(P(y)\). Knowing only \(P(y|x)\) is not enough to recover all of these quantities.

This is the theoretical reason behind the asymmetry. A generative model tries to learn a richer object, so classification becomes one consequence of that knowledge.

This same asymmetry is not limited to Gaussian toy models. It also appears in modern generative models, where the distributions are learned by neural networks instead of simple formulas.
