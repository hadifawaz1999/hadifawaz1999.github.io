3. A Synthetic Experiment

To make the distinction concrete, let us create a simple two-dimensional dataset with two classes.

$$ x = (x_1, x_2), \qquad y \in \{0,1\} $$

Each class is generated from a different Gaussian distribution. This gives us two visible clouds of points.

The discriminative model

The discriminative model directly learns the probability of the label given the observation:

$$ P(y|x) $$

In our experiment, we use logistic regression:

$$ P(y=1|x) = \sigma(w^T x + b) $$

The decision boundary appears when the model is equally uncertain between the two classes:

$$ P(y=1|x)=0.5 \quad \Longrightarrow \quad w^T x + b = 0 $$

Because this equation is linear in \(x\), the discriminative boundary is a straight line.

The generative model

The generative model learns something different. Instead of directly learning \(P(y|x)\), it learns one distribution per class:

$$ P(x|y=0) \qquad \text{and} \qquad P(x|y=1) $$

In this experiment, each class is modeled as a Gaussian:

$$ P(x|y=c) = \mathcal{N}(\mu_c,\Sigma_c) $$

The model estimates the mean \(\mu_c\), covariance \(\Sigma_c\), and prior probability \(P(y=c)\) for each class.

How the generative boundary appears

For a new point \(x\), the model computes how likely the point is under each class distribution:

$$ P(x|y=0)P(y=0) \qquad \text{versus} \qquad P(x|y=1)P(y=1) $$

The point is assigned to the class with the larger score.

$$ \hat{y} = \arg\max_{c \in \{0,1\}} P(x|y=c)P(y=c) $$

The decision boundary is the set of points where both class scores are equal:

$$ P(x|y=0)P(y=0) = P(x|y=1)P(y=1) $$

This is the crucial difference: the generative model does not directly learn the boundary. The boundary emerges from comparing the learned class distributions.

Since the two classes can have different covariance matrices, \(\Sigma_0 \neq \Sigma_1\), the resulting boundary is generally quadratic:

$$ x^T A x + b^T x + c = 0 $$

This is why the generative boundary can bend, while logistic regression produces a straight line in this experiment.

Generation as an extra capability

Once the class distributions are learned, the model can also create new samples by sampling from them:

$$ x_{\text{new}} \sim P(x|y=c) $$
The experiment shows the asymmetry clearly: the generative model can derive a classifier from distributions, but the discriminative model cannot derive distributions from a boundary.
