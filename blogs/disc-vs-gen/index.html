Discriminative vs Generative Models

Why learning to classify data is not the same as learning how data is generated.

Ali Ismail-Fawaz June 2026 12–15 min read Intermediate
Machine Learning Probability Bayes Rule Generative Models

The central question

A discriminative model learns how to separate classes. A generative model tries to learn how the data itself could have been produced.

This difference is often summarized as “one classifies and one generates,” but that explanation is too shallow. In this article, we will look at the distinction through probability, geometry, and a synthetic experiment.

If a model understands how each class generates data, classification can emerge from that knowledge. But if a model only learns the classification boundary, generation does not naturally follow.
