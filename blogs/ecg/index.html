About the Demo

Designed for visual ECG learning

The visualization plays back ECG recordings and highlights simplified heart contraction patterns in sync with ECG phases: P, QRS, and T. It is intended for demonstration and educational use.

How to use

  • Press Play to start playback.
  • Use Pause to stop playback at the current time.
  • Press Restart to return to the beginning.
  • Select another Record to load a different ECG sample.
  • Use Speed to change playback rate.

Dataset

Source: Lobachevsky University Electrocardiography Database

Sampling rate: 500 Hz

Notes

  • The heart animation is a simplified timing illustration, not a diagnostic model.
  • ECG phase intervals come from annotations provided by the original dataset.
